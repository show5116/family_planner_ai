# AI 에이전트 도메인 지식 (Agent Domain Knowledge)

이 문서는 AI Agent가 Family Planner 시스템을 이해하고 작업을 수행하기 위한 **핵심 도메인 규칙**과 **가용 능력(Tools/APIs)**을 정의합니다. 백엔드 구현 상세가 아닌, 에이전트가 "무엇을 할 수 있는지"에 초점을 맞춥니다.

## 1. 핵심 개념 (Core Concepts)

### 🌍 환경 (Ecosystem)
- **사용자(User)**: 1명의 사용자는 여러 **그룹(Group)**에 속할 수 있습니다.
- **그룹(Group)**: 가족, 친구, 모임 등 일정과 자산을 공유하는 단위입니다.
- **역할(Role)**: 그룹 내에서 `OWNER`(방장), `ADMIN`, `MEMBER` 권한을 가집니다.

### ✅ 일정 및 할 일 (Tasks)
Family Planner는 **일정(Calendar)**과 **할 일(Todo)**을 통합 관리합니다.
- **통합 모델**: 하나의 Task 객체로 일정과 할 일을 모두 표현합니다.
- **유형(Type)**:
    - `CALENDAR_ONLY`: 시간 점유가 중요한 일정 (예: 회의, 여행)
    - `TODO_LINKED`: 완료 체크가 중요한 할 일 (예: 장보기, 청소)
- **반복(Recurring)**: 매일/매주/매월 등 스마트 반복 설정이 가능합니다. (예: "3주마다 토요일")
- **이중 날짜**: `scheduled_at`(시작일)과 `due_at`(마감일)을 가집니다.

### 👶 육아 포인트 (Childcare)
자녀의 생활 습관을 관리하는 보상 시스템입니다.
- **포인트/용돈**: 칭찬(Reward)과 벌칙(Penalty) 규칙에 따라 포인트가 증감합니다.
- **적금(Savings)**: 자녀가 모은 포인트를 적금에 넣고 이자를 받을 수 있습니다.

### 🔔 알림 (Notifications)
- **즉시 알림**: 채팅, 초대 등 실시간 전송.
- **예약 알림**: "30분 뒤에 알려줘"와 같은 스케줄링이 가능합니다.

## 2. 에이전트 능력 (Agent Capabilities)

에이전트는 백엔드 API를 통해 다음과 같은 행동을 수행할 수 있습니다.

### 📅 일정 관리 (Planning)
| 행동 | 설명 | 필요한 정보 |
| :--- | :--- | :--- |
| **일정 조회** | 특정 기간/조건의 일정을 찾습니다. | 날짜 범위, 그룹 ID, 카테고리 |
| **일정 생성** | 새 일정이나 할 일을 등록합니다. | 제목, 날짜, 유형, 반복 여부 |
| **참여자 지정** | 일정에 특정 멤버를 태그합니다. | Task ID, User ID 목록 |
| **빈 시간 찾기** | 가족 구성원들의 일정을 피해 빈 시간을 찾습니다. | 대상 그룹, 기간 |

### 🏘️ 그룹 관리 (Community)
| 행동 | 설명 | 필요한 정보 |
| :--- | :--- | :--- |
| **그룹 목록** | 사용자가 속한 그룹들을 확인합니다. | User ID |
| **초대 코드** | 그룹 초대를 위한 코드를 생성/조회합니다. | Group ID |

### 💰 육아/보상 (Childcare)
| 행동 | 설명 | 필요한 정보 |
| :--- | :--- | :--- |
| **포인트 지급** | 자녀에게 보상/용돈을 줍니다. | 대상 자녀, 금액, 사유 |
| **적금 관리** | 자녀 계좌 입/출금 처리를 돕습니다. | Account ID, 금액 |

### 📮 알림 (Messaging)
| 행동 | 설명 | 필요한 정보 |
| :--- | :--- | :--- |
| **예약 발송** | 특정 시간에 알림을 보내도록 예약합니다. | 메시지, 발송 시간, 대상 |

## 3. 데이터 모델 요약 (Simplified Schema)

에이전트가 추론에 참고해야 할 데이터 구조입니다.

### **Task**
```json
{
  "id": "uuid",
  "title": "가족 저녁 식사",
  "type": "CALENDAR_ONLY | TODO_LINKED",
  "priority": "LOW | MEDIUM | HIGH | URGENT",
  "scheduledAt": "ISO-8601",
  "dueAt": "ISO-8601",
  "isCompleted": false,
  "participants": ["user_id_1", "user_id_2"],
  "recurring": {
    "interval": 1, 
    "type": "WEEKLY",
    "days": ["SAT"]
  }
}
```

### **ChildcareRule**
```json
{
  "name": "방 청소 안함",
  "penalty": 500,
  "isActive": true
}
```
